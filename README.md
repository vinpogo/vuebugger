# Vue utilities

## Vuebugger

Vue Devtools helper for debugging composables and reactive state. Add `debug()` to expose values in Devtools without production overhead.

- Package: [packages/vuebugger](packages/vuebugger)
- Docs: [packages/vuebugger/README.md](packages/vuebugger/README.md)

Install:

```bash
pnpm add -D @vingy/vuebugger
```

## Vueltip

Headless tooltip plugin for Vue with smart visibility detection and a single shared tooltip instance.

- Package: [packages/vueltip](packages/vueltip)
- Docs: [packages/vueltip/README.md](packages/vueltip/README.md)

Install:

```bash
pnpm add vueltip
```

## Demo

Try both packages in the demo app:

```bash
pnpm install
pnpm demo
```

Open the demo from [demo/](demo/).
